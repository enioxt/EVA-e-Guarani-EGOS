# EVA & GUARANI Sandbox Frontends

This directory contains different frontend interfaces for the EVA & GUARANI sandbox environment.

## Available Frontends

- **html_basic**: A simple HTML, CSS, and JavaScript implementation that works directly with the Flask API.
- **react_app**: A React-based implementation with more interactive features.
- **vue_app**: A Vue.js implementation with advanced visualization capabilities.

## Getting Started

Choose the frontend that best suits your needs and follow the instructions in its respective directory.

The basic HTML frontend requires no build steps and works immediately with the sandbox API.
